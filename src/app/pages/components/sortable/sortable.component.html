Sortable

The sortable component represents a list of items, with ability to sort them or move to another container via drag&drop. Input collection isn't mutated by the component, so events ngModelChange, onChange are using new collections.

NOTE: The Sortable Component does not have a native equivalent in Foundation for Sites, so we used callout containers with some minor styling as a default.

Usage

Examples

Basic

Re-arrange the list or move from one column to the other.


model: {{ itemStringsLeft | json }}

model: {{ itemStringsRight | json }}

Complex Data Model

Re-arrange the list or move from one column to the other.


model: {{ itemObjectsLeft | json }}

model: {{ itemObjectsRight | json }}

Custom Item Template

Re-arrange the list or move from one column to the other.

{{index}}: {{item.value}} 

model: {{ customStringsLeft | json }}

model: {{ customStringsRight | json }}
