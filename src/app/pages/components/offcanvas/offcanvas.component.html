Off-Canvas

Off-canvas panels are positioned outside of the viewport and slide in when activated.

The Off-Canvas component uses Zurb's Motion UI Sass library for the animations, so be sure to import Motion UI into your stylesheet if you want animations:

Wrapper

You can add an optional wrapper around off-canvas and the content. This hides the vertical (on top/bottom off-canvas) or horizontal (on left/right off-canvas) scrollbars the body will have. Simply add a container with the class .off-canvas-wrapper.
