Callouts

Callouts combine panels and alerts from Foundation 5 into one generic container component.

NOTE: Callouts are the equivalent to Bootstrap Alerts and use the AlertModule.

Usage

Examples

Basic

Alerts are available for any length of text, as well as an optional dismiss/close button. The 'callout' class is automatically added for you, but for proper styling use one of the five required contextual classes (i.e. primary, secondary, success, warning, and alert).
