Basic Button Dropdown Menu

Any sized button can become a dropdown toggle with few markup changes.

Dropdown widths are globally set by the $dropdown-width variable in your _settings.scss file.
If you want to display the dropdown caret icon be sure to add the .dropdown class to your button.

Basic Dropdown Pane

To create a dropdown pane, simply add the class .dropdown-pane to the content section of your dropdown. Adding the .dropdown-pane class gives the container proper padding and limits the width to whatever you have set in your variable $dropdown-width.

Manual Triggering

Dropdowns can be triggered by show, hide and toggle methods from directive. Use method toggle(true) if you want to toggle the dropdown or toggle(false) if you want to only close opened dropdown.
