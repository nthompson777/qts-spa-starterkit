Progressbar

Provide up-to-date feedback on the progress of a workflow or action with simple yet flexible progress bars.

NOTE: ngx-foundation adds CSS based animations $progress-meter-animation to the progressbars and fixes the progress-meter-text alignment when the percentage text has a low value (if displaying with text).

If you wish to remove animations, simply set the variable to none $progress-meter-animation: none;. It is recommended to use a darker color for the $progress-background or darken the default gray $progress-background: darken($medium-gray, 8%); so low percentages text is visible on top of the progressbar background.
